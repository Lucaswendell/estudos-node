# Anotações
### Rodar o node
```bash
$ node arquivo.js
```
### Gerar paths.md
```bash
$ cd readfolder && npm start
```
### Gerar package.json rapido
```bash
$ npm init -y
```
### Denpencias DEV
```bash
$ npm install <pkg> --save-dev
```
### Atualizar pacotes
```bash
$ npm update
$ npm update <pkg>
$ npx npm-check-updates
```
### Pacote global
```bash
$ npm install <pkg> -g
```
### Desistalar pacote
```bash
$ npm uninstall <pkg>
```
